class viam.components.imu.IMU(name: str)[source]

IMU represents a physical IMU (inertial measurement unit).

This acts as an abstract base class for any drivers representing specific IMU implementations. This cannot be used on its own. If the __init__() function is overriden, it must call the super().__init__() function.

class viam.components.imu.EulerAngles(*, roll_deg: float = ..., pitch_deg: float = ..., yaw_deg: float = ...)

EulerAngles are three angles used to represent the rotation of an object in 3D Euclidean space The Tait–Bryan angle formalism is used, with rotations around three distinct axes in the z-y′-x″ sequence.
