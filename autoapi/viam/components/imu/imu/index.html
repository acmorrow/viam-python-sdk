class viam.components.imu.imu.IMU(name: str)[source]

IMU represents a physical IMU (inertial measurement unit).

This acts as an abstract base class for any drivers representing specific IMU implementations. This cannot be used on its own. If the __init__() function is overriden, it must call the super().__init__() function.
