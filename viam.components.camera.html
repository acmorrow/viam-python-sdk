viam.components.camera.camera module

class viam.components.camera.camera.Camera(name: str)

Bases: viam.components.component_base.ComponentBase

Camera represents any physical hardware that can capture frames.

This acts as an abstract base class for any drivers representing specific camera implementations. This cannot be used on its own. If the __init__() function is overriden, it must call the super().__init__() function.
