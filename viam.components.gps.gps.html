viam.components.gps.gps module

class viam.components.gps.gps.GPS(name: str)

Bases: viam.components.component_base.ComponentBase

GPS represents a GPS unit that can report latitude/longitude, altitude, and speed measurements.

This acts as an abstract base class for any drivers representing specific GPS implementations. This cannot be used on its own. If the __init__() function is overriden, it must call the super().__init__() function.
